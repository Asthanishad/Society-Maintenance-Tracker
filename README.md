# 🏢 Society Maintenance Tracker

A full-stack MERN web application for managing society maintenance activities. The system provides separate dashboards for Admin and Residents to manage notices, complaints, maintenance bills, and user profiles.

---

## ✨ Features

### 👨‍💼 Admin
- Admin Login
- Dashboard
- Create / Delete Notices
- Pin Important Notices
- View & Manage Complaints
- Update Complaint Status
- Create Maintenance Bills
- Delete Maintenance Bills
- View Residents
- Profile Management

### 🏠 Resident
- Resident Registration & Login
- Dashboard
- View Notices
- Register Complaints
- Upload Complaint Images
- Pay Maintenance Bills
- View Payment History
- Profile Page

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Asthanishad/Society-Maintenance-Tracker.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
